HUMAN-ACTION — execute after smoke tests pass.

**Pre-condition:** DNS TXT record `google-site-verification=<value>` must be in Cloudflare DNS (was set in Plan 10-04 Task 2).

1. Open Google Search Console (https://search.google.com/search-console):

2. Add a Domain Property:
   - Click "Add property" → Select "Domain" type
   - Enter: `proactivsports.com` (no protocol prefix)
   - Click Continue

3. GSC shows the TXT record value to add. This should already be in Cloudflare DNS (set in Plan 10-04). Confirm by running:
   ```bash
   dig +short TXT proactivsports.com | grep "google-site-verification"
   ```
   If not present: add to Cloudflare immediately and wait 5–15 minutes before clicking Verify.

4. Click "Verify" in GSC. If verification succeeds → proceed. If it fails → wait 15–30 minutes and retry (TXT propagation can take time even after NS switch).

5. After Domain Property is verified, submit all three sitemaps:
   - GSC → Sitemaps → New sitemap → `https://proactivsports.com/sitemap.xml` → Submit
   - GSC → Sitemaps → New sitemap → `https://hk.proactivsports.com/sitemap.xml` → Submit
   - GSC → Sitemaps → New sitemap → `https://sg.proactivsports.com/sitemap.xml` → Submit

6. Confirm sitemaps show "Success" status (may take a few minutes; "Pending" is normal immediately after submission).

Note: A Domain Property for `proactivsports.com` automatically covers hk. and sg. subdomains — all subdomain data flows into the same property.
